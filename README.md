## 🌟 Overview

AuraCalc is a **premium all-in-one calculation suite** designed to simplify everyday complex math. From financial planning to health tracking, it delivers a hardware-accelerated experience with a stunning **3-way theme engine** — Light, Dark, and System sync.

> 💡 Built for students, professionals, and anyone who needs fast, accurate calculations in a beautiful interface.

🔗 **Live Demo:** [aura-calc.vercel.app](https://aura-calc.vercel.app)
📦 **GitHub:** [Jaanvichouhan34/AuraCalc](https://github.com/Jaanvichouhan34/AuraCalc)

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/Jaanvichouhan34/AuraCalc.git
cd AuraCalc/frontend
```

**2. Install dependencies**
```bash
npm install
```

**3. Start development server**
```bash
npm run dev
```

**4. Build for production**
```bash
npm run build
```

Open [http://localhost:5173](http://localhost:5173) in your browser 🎉
